== Description ==

The Customizer Login UI plugin allows you to create a customized login page for your WordPress website. If you need to personalize your basic admin login page, this plugin is helpful.

== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **Customizer Login UI**.
3. Install and activate the Customizer Login UI plugin.

= Manual installation =

1. Upload the entire `customizer-login-ui` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Customizer Login UI plugin.
